### How to Install eLearning Theme?

Installing an eLearning theme is quite easy. You can install the theme on your WordPress site basically using two methods, i.e., **WordPress Dashboard** and **Upload Method**. 

Let's go into detail about how to install an eLearning theme. 

### Installing eLearning theme via WordPress Dashboard 

It is one of the simplest and fastest ways to install the eLearning theme on your WordPress Dashboard on your site. Follow the instruction given below:

- Navigate to your **WordPress Dashboard > Appearance > Themes** and click on the Add New. 

- From the search functionality on the WordPress dashboard, search for the eLearning theme. 

- Click on **Install** and **Activate** it. 

It is shown in the image below: 

![install-theme-via-dashboard](https://user-images.githubusercontent.com/109593521/183350959-0e117ed8-b5f4-4fdd-9e36-71b82c521553.png)

### Installing eLearning theme via Upload Method (Alternative Method) 

The upload method is suitable if you want to install the already downloaded eLearning theme file. In this method, you will need to upload the eLearning theme from your local computer to the WordPress dashboard. Follow the steps below:

- Go to [Masteriyo elearning page](https://masteriyo.com/wordpress-lms/themes/elearning/)

- Click the Download button on the eLearning page.

- Go to **Appearance > Themes** and go to Add New. Click on Upload Theme and upload a downloaded **elearning theme file**.

- Click on Install Now and Activate it.

![install-theme-upload](https://user-images.githubusercontent.com/109593521/183344728-e909c05a-3b44-4051-942d-3f494fbf3721.gif)
