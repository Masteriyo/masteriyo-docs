### How to Install eLearning Theme? - Free version: 

Installing an eLearning theme is quite easy. You can install the theme on your WordPress site basically using two methods, i.e., **WordPress Dashboard** and **Upload Method**. 

Let's go into detail about how to install an eLearning theme. 

### Installing eLearning theme via WordPress Dashboard 

It is one of the simplest and fastest ways to install the eLearning theme on your WordPress Dashboard. Follow the instruction given below to install the eLearning theme. 

- Navigate to your **WordPress Dashboard > Appearance > Themes** and click on Add New. 

- From the search functioâ‚¬nality on the WordPress dashboard, search for the eLearning theme. 

- Click on **Install** and **Activate** it. 

It is shown in the image below: 

![install-theme-via-dashboard](https://user-images.githubusercontent.com/109593521/183350959-0e117ed8-b5f4-4fdd-9e36-71b82c521553.png)

### Installing eLearning theme via Upload Method (Alternative Method) 

The upload method is suitable if you want to install the already downloaded eLearning them. In this method, you will need to upload the eLearning theme from your local computer to the WordPress dashboard. Follow the steps below to install the eLearning theme 

- [Click on this link](https://downloads.wordpress.org/theme/elearning.zip) and a file named **elearning.zip** will automatically be downloaded to your computer. 

- Go to **Appearance > Themes** and go to Add New. Now click on Upload Theme and upload a downloaded **elearning.zip** file on it. 

- Click on Install Now and Activate it.
