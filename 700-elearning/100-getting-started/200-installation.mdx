### How to Install eLearning Theme? - Free version: 

eLearning theme is really simple to install. You can instal the theme on your WordPress site basically by using two methods i.e., WordPress Dashboard and Upload Method. 

Let's go into detail about how to install eLearning theme. 

### Installing eLearning theme via WordPress Dashboard 

It is one of the simplest and fastest ways to install the eLearning theme on your WordPress Dashboard. Follow the instruction given below to install the eLearning theme. 

- Navigate to your WordPress Dashboard > Appearance > Themes and click on Add New. 
- From the search functionality on the WordPress dashboard, search for the eLearning theme. 
- Click on Install and Activate it.

### Installing eLearning theme via Upload Method (Alternative Method) 

The upload method is suitable if you want to install the already downloaded eLearning them. In this method, you will need to upload the eLearning theme from your local computer to the WordPress dashboard. Follow the steps below to install the eLearning theme 

- [Click on this link and a file named elearning.zip will automatically be downloaded to your computer. 
- After the download is completed, look for the elearning.zip file on your computer. It will likely be in your Downloads folder. Once you find it, remember the location and continue to the next step. 
- Go to **Appearance > Themes** and go to Add New. Now click on Upload Theme. 
- Click on browse and navigate to the downloaded elearning.zip file and click on Install Now. 
- Activate it.

### Activate eLearning Theme 

In case if you missed activating the theme deactivate the eLearning theme, you can activate it manually with the following steps: 
 
- Go to Appearance > Themes in the WordPress Dashboard. 
- Hover over the eLearning theme thumbnail and click on the Activate button. 
- The theme is activated successfully
