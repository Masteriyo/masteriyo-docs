### Introduction: 

Thank you for choosing the eLearning Theme! 

**eLearning Theme** is a simple and clean new free WordPress Learning Management System (LMS) theme powered by Masteriyo.  Masteriyo is a revolutionary LMS plugin built on React JS which is not only super-fast but also super easy to setup. Masteriyo plugin makes it simple to design and sell online courses
