# How to add social icons/links on header?

It is possible to add social icons with link in the top header bar or any other widget area as instructed below:

- Navigate to any of the text editor available in the eLearning theme. For example: Navigate to Header Top Bar and choose Text/HTML from Left Content or Right Content drop-down.

- Select text from the textarea tab.

- Add the following code inside the textarea box:

```
<div class="header-bar-social-icons">
    <a href="https://facebook.com/elearningtheme/">
        <i class="fa fa-facebook"></i>
    </a>
    <a href="https://twitter.com/elearningtheme/">
        <i class="fa fa-twitter"></i>
    </a>
    <a href="https://plus.google.com/elearningtheme">
        <i class="fa fa-google-plus"></i>
    </a>
    <a href="https://linkedin.com/elearningtheme">
        <i class="fa fa-linkedin"></i>
    </a>
</div>

```

- In the above code, edit the link and [code for FontAwesome ](https://fontawesome.com/v4/icons/)Icons as per your preference.
