# How to change the main Header Banner image from demos?

After importing the eLearning Demo using our Demo Importer Plugin, you might find it difficult to change the main Header Banner image on the homepage.

To change the image, you can follow these steps:

- **Edit the Homepage with Elementor**: It can be done in various ways, one of which is by clicking Edit with Elementor on your Home Page.

It is shown in the image below:

![Change Header Banner Image](https://user-images.githubusercontent.com/109593521/183626734-f6c077cd-2683-4021-976f-bebef27cdb0a.png)



    For more details, you can consult this article: How to edit Elementor pages in eLearning Theme?


- On the Elementor edit page, **Right Click** near the corner of the image and click on **Edit Section** or just click on **section handle**.

It is shown in the image below:

![Edit Section](https://user-images.githubusercontent.com/109593521/183627157-a9eb88d1-5b32-4edf-b8ed-5c4b73b0a20a.png)

**OR**

![Edit Section-Alternative](https://user-images.githubusercontent.com/109593521/183627515-37b9f13f-de1b-455e-a31b-bf793029b568.png)


- Clicking on **Edit Section** opens a panel on the left. Go there and click on the **Style** tab on the top.

- Look for the **Image** option under the **Background** Section. You can click on the image below to choose a new one for the Header Banner.

It is shown in the image below:

![Edit Header Image](https://user-images.githubusercontent.com/109593521/183628441-c83a2bd4-c231-4d29-a836-9911f07cc0ea.gif)
