# How to Add a Site Icon?

To upload a site Icon, follow the steps as mentioned below:

- From your **WordPress Dashboard**, go to the **Appearance > Customize > Header > Site Identity** section.

- Find the **Site Icon** option. Click on Select site icon and select an icon.

- Click on **Publish**.
