eLearning theme allows you to manage site title, logo, tagline and others in Site identity section. 

- From Customize section, go to the **Header > Site Identity**.
- In Site identity option, you will see  available options
  - Logo
  - Retina Logo
  - Site Icon
  - Site Title
  - Tagline
  - Colors 
    - Header Text Color 
  - Typography
    - Site Title
    - Site Tagline
