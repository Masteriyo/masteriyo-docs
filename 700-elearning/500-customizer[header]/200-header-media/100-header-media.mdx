# How to change the Header Media?

This theme supports the Header Media options that have been provided by WordPress.

- Go to **Appearance > Customize > Header > Header Media** section.
- Click on Add new image/ video to upload the required image/video for your header.
- Click on **Publish**.

```
Note: This feature can be used to get a transparent header above your image as shown below. To do so, just enable the Transparent Header option from the Page Settings.
```
