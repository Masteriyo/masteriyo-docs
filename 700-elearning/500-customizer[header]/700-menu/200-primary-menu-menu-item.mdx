# How to customize the Primary Menu-Menu Item of your site?

      Location: Appearance > Customize > Header > Menu > Primary Menu : Menu Item

You have additional customization options for items in your Primary Menu in the eLearning. The options are as follows:

- **Active Menu Item Style**: You can pick from one of the four style options for your Active menu items.
  
  - None

  - Underline Border

  - Left Border

  - Right Border

![Active Menu Item Style](https://user-images.githubusercontent.com/109593521/183563729-6983f733-b485-4268-9ca3-37c1b0de67e5.png)


- **Colors**: You can choose colors for Menu items
  
  - Menu Item
    
    - Normal
    
    - Hover
    
    - Active

![Primary Menu Item - Colors](https://user-images.githubusercontent.com/109593521/183563951-b981c475-950a-44d3-8ec9-08aa6222293c.png)

- **Typography**: Under the Typography option, choose the customization for Primary Menu fonts such as Font Family, Weight, Font Size, Line Height, Style, and Transform.

![Primary Menu Item-Typography](https://user-images.githubusercontent.com/109593521/183564850-0b7967c7-a4c8-45e8-87b0-047af3477d37.png)
