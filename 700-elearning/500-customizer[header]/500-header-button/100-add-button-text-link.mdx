# How to add button text and link on the header?

- From your WordPress Dashboard, go to **Appearance > Customize > Header > Header Button** section.
- Available Options
   - **Button Text**: Add your text in the **Button Text** field.
   - **Button Link**: Insert the link in the **Button Link** field in order to redirect the button to the link.
   - **Open link in a new tab**: This option can be used to open the links in a new tab when the button is clicked.

- Click on **Publish**.

*```Note: You need to have assigned Primary Menu to display the header button.```*
