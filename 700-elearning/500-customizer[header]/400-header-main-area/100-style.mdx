# How to manage the style option from the Header Main Area?

You can change the position of **Site Identity** and **Navigation Menu** on the Header Main Area using the **Style** option. Choose the **Style** from available options like below:

- **First option**: Site Identity on the **Left** & Navigation Menu on the **Right** side.
- **Second option**: Both at the **Centre**.
- **Third option**: Site Identity on the **Right** & Navigation Menu on the **Left** side.
