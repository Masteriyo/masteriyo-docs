# How to change the color and design of the menu on a specific page?

WIth the eLearning theme, you can change the color and design of the Menu in any specific page.

Just follow these steps:

- From your **WordPress Dashboard**, go to the **Pages > All Pages** section.

- Choose the required **Page** you want to customize.

- Below the editor, go to the **Page Settings** option.

- Under the Page Settings option, go to the **Primary Menu** option.

The available options for customizing the Primary Menu using the Page Settings are as follows:

- **Menu Item Color**: Select the color for the menu item.

- **Menu Item Hover Color**: Select the color for the menu item on the Hover state.

- **Menu Item Active Color**: Select the color for the Active menu item.

- **Active Menu Item Style**: Choose from the dropdown menu to pick the style for the Active menu item. You can leave it as Default to keep the settings from the Customizer.
