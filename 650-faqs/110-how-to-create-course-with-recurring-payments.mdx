Follow the steps below to create a subscription based course with recurring payments. 

**Requires**

1. [Masteriyo Pro Plugin](https://masteriyo.com/pricing/)
2. [Stripe Payment Gateway Add-on](https://docs.masteriyo.com/getting-started/activate-masteriyo-addons) 

## Configuration

Once you have set up the Stripe Payment Gateway for your courses, you can create a course with subscription by following the steps below.

- Go to Courses > Select a Course.
- Go to Settings > Pricing

In the course that has payment set as **Paid**, you can choose **Recurring** 

![recurring-payments](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/123ab20c-2683-4ab9-8cec-0bba1013304a)

Here, you can set the time period for the subsciption along with the subsciption fees. You can set the expiry time in months. The maximum expiry time can be set upto 24 months.
