title: How to create Facebook App ID and secret?
show_child_cards: true

---

This documentation will help you generate the Facebook App ID and secret required for integrating the Masteriyo Social Login add-on.

- Go to [Facebook Developer Console](https://developers.facebook.com/).
- Login to your Facebook account and click on **Create App** button.

You can now use these credentials to integrate Google Login on the Masteriyo. Please note you have activated the Social Login add-on.
