---
title: How to edit a course with WordPress Editor?
show_child_cards: true
excerpt: You can edit a course with WordPress Editor in Masteriyo.
---

To edit any course with the WordPress Editor, go to the following location.

        Masteriyo > courses

Beside every course, you can see a button to show options. Click on this option and click on WordPress Editor.

![wordpress-editor-for-course](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/c918f700-9666-4ab0-b9dc-6d5b47ff3eac)

Now, all the blocks you add will be shown on the Course Overview.

![block-example](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/11015d02-f294-4012-bcd3-cab06bd8f87c)
