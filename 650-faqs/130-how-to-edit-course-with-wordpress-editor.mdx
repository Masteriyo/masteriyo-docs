---
title: How to choose editor for creating/editing a course?
show_child_cards: true
excerpt: You can edit a course with default/block editor in Masteriyo.
---

In order to select the desired editor, please go to the following location.

        Masteriyo > Settings > General > Editor

![main-setting](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/a7079434-5b89-4317-bb36-bc1ac2c18c7d)

Once you have selected the editor and saved the changes, we suggest you reload the page to reflect the changes.

Now, you can edit/create the courses with the selected choice of editor.

![wordpress-editor-for-course](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/c918f700-9666-4ab0-b9dc-6d5b47ff3eac)

**Course Details Editing**

![course-overview](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/72fb6803-a822-434e-b990-9b4b68aaf846)
