You can edit the courses with the WordPress Editor to get access of all the features that might have been restricted in the Block editor.

- Go to the following location:

        Masteriyo > Courses

- Click on the 'three dots' option beside the Edit button of the respective course.

![course-wordpress-editor](https://github.com/user-attachments/assets/bcdc8c11-aee9-4aef-8a3d-30d5a7c0216a)

- Click on the WordPress Editor.

Now, you can edit the courses with the WordPress Editor.

![wordpress-editor](https://github.com/user-attachments/assets/4ef32e55-6eab-4fd7-8365-151dd0e8000d)
