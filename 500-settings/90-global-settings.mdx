To go to the global settings of the Masteriyo plugin, go to the following location.

        Location: Masteriyo > Settings

![MainLocation](https://user-images.githubusercontent.com/75772539/229495647-c4f1440f-7e82-4369-9790-9fabcb791933.png)

## General

### Pages

![General Pages](https://user-images.githubusercontent.com/75772539/228783902-459717e4-591b-4bc7-867d-f8b764fa242c.png)

- **Course Page** - Select the page where you want to show all the courses collectively.
- **Learn Page** - Select the learn page.
- **Account Page** - Select the page for the user's account.
- **Checkout Page** - Select the page to redirect the users for checkout.
- **Instructor Registration Page** - Select the page for the instructors to register.

### Styling

![General-Styling](https://user-images.githubusercontent.com/75772539/228784157-f948d167-ded2-46f6-ab94-0faf70a1e496.png)

- **Primary Color** - Select the color that matches your brand/site.
- **Primary Color for Learn Page** - Select the color to reflect on the Learn Page.
- **Button Color** - Select the color to appear on the buttons.

## Course Page

![Course-page](https://user-images.githubusercontent.com/75772539/228784454-02a59f57-f630-4ebe-998e-a2c71ed7b937.png)

- **Show Search** - Enable/Disable search bar on the courses page.
- **Courses Per Page** - Set the number of courses to show on a page.
- **Courses Per Row** - Set the number of courses to appear on a row.

## Single Course Page

### Display

![Single-Course-Page-Display](https://user-images.githubusercontent.com/75772539/228784702-400353d9-d8fb-438e-8206-af1f2dfd3096.png)

- **Enable Review** - Enable this option to allow the students to review the course

### Related Course

![Single-Course-Page-Related-Course](https://user-images.githubusercontent.com/75772539/228785353-e4e6a536-f76b-47ba-89f9-7f33bd299836.png)

- **Enable** - Option to enable the related course section
- **Related Attribute** - Select either category or instructor as the attribute for the related courses section.
- **Maximum Related Course** - Set the maximum amount of courses to appear on the related courses section.

## Payments

### Store

![Payment-Store](https://user-images.githubusercontent.com/75772539/228786093-fc04114e-b463-483a-9694-32f705fbf5de.png)

- **Country** - Select the country where your store is based on.
  **Other Address Fields For the Store**
- **State**
- **City**
- **Address Line 1**
- **Address Line 2**

### Currency

![Payment-Currency](https://user-images.githubusercontent.com/75772539/228786357-7ef5097a-116a-44d3-b4e8-5d39810dd58a.png)

- **Currency** - Select the currency you want to use for the courses.
- **Currency Position** - Select the position(left/right) to display the currency symbol.
- **Thousand Separator** - Set a character for the number places.
- **Decimal Separator** - Set a character for decimal numbers.
- **Number of Decimals** - Set the decimal number place.

### Standard Paypal

![Payment-Standard](https://user-images.githubusercontent.com/75772539/228786973-3f76bb7a-037c-4d57-be4b-c7b16e92ada7.png)

- **Enable** - Enable this option to use the Paypal payment method.
  - **Title** - Write the title of the payment for it to be seen in the Checkout page.
  - **Description** - Describe the payment method.
  - **IPN Email Notification** - Enable this to send a notification when IPN is received from Paypal.
  - **Paypal Email** - Email used in your paypal account.
  - **Receiver Email** - If your main Paypal email is different from the above entered email, set it here.
  - **Identity Token** - Add the identity token. An identity token in PayPal is a unique alphanumeric code used for secure authentication and integration with third-party applications or services.
  - **Invoice Prefix** - Provide a prefix for the invoice.
  - **Payment Actions** - Select either Capture or Authorize as the payment actions.
  - **Image URl** - URL for a 150x50px image to be displayed on the checkout page as the logo.
  - **Debug Log** - Enable this option to store debug logs.
  - **Sandbox** - Enable this if you want to use test/sandbox account.
  - **Live API username**
  - **Live API password**
  - **Live API Signature**

### Offline Payments

![Payments-Offline-Payments](https://user-images.githubusercontent.com/75772539/228787408-c7f6e73a-1702-4da6-9122-d13a68f74d5f.png)

- **Enable** - Enable this option to use offline payment method during checkout.
  - **Title** - Provide the title to the offline payment.
  - **Description** - Describe the offline payment method.
  - **Instruction** - Specify if you have any instructions required.

### Stripe

![Payments-Stripe](https://user-images.githubusercontent.com/75772539/228787886-67a69ffe-9b9b-444d-9861-07d9bf9313b1.png)

- **Enable** - Enable this option to use the Stripe payment method.
  - **Title** - Provide the title to the Stripe payment.
  - **Description** - Describe the payment method.
  - **Sandbox** - Enable this option if you want to use the sandbox/test Stripe payment.
  - **Test Publishable Key** - Get you API credentials from Stripe and add the test publishable key here.
  - **Test Secret Key** - Add the test secret key from the Stripe here.
  - **Webhook Secret Key** - Get your webhook secret key from Stipe and add here.
  - **Webhook Endpoint** - Add the given endpoint to your Stripe webhook endpoint list to verify the payment status.

## Emails

![Email Settings](https://user-images.githubusercontent.com/75772539/228789048-d82ef8ba-7d8c-41dd-8853-852836aaeed0.png)
