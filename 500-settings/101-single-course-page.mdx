### Display

![singlecoursepage-display](https://github.com/user-attachments/assets/b66dc453-df4d-4e92-93eb-e9988e1105dc)

- **Enable Review** - Enable this option to allow the students to review the course.
- **Review for Enrolled Users Only** - Enable this to display review tab for enrolled users only.
- **Enable Course Visibilty** - Enable this option to force the users to log in to view paid courses.

### Related Course

![singlecoursepage-related-course](https://github.com/user-attachments/assets/7f5e985d-ba1e-45ca-a752-bbf1694444bd)

- **Enable** - Option to enable the related course section.
- **Related Attribute** - Select either category or instructor as the attribute for the related courses section.
- **Maximum Related Course** - Set the maximum amount of courses to appear on the related courses section.

### Social Share - [Pro]

![singel-course-page-social-share](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/6061a2ad-b758-4a1c-8eec-e4adf7e9dcbf)

You can choose social sites to allow users to share the course. You can change the given design attirbutes for the social share icons.

### Template

![singlecoursepage-template-choose-layout](https://github.com/user-attachments/assets/7a0e44a5-06fd-46cd-9403-604951d5253e)

You can create an Elementor Template and use it in the single course page or use the default Masteriyo course page template as per your needs. For this, enable the custom template, select the source of the template from 'Template From' and select the template from the next option.

![singlecoursepage-template-custom-template](https://github.com/user-attachments/assets/7159845f-7fba-4581-ba13-487955000fe4)
