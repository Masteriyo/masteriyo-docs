## Learn Page

### General

![learn-page-general](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/eeabab19-c39a-4fe4-b8aa-25a6efeb7315)

- **Logo** - Upload your brand's logo.
- **Auto Load Next Content** - Enable this option to automatically navigate to the next content as we press 'Mark as Complete'.

### Display

![learn-page-display](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/a6cf11cf-54eb-4cc2-91a4-3670476ff52e)

- **Enable Questions & Answers** - Enable this option to display questions and answers tab in the learn page.
- **Show Sidebar Initially** - When you enable this, the page will navigate to next content as you press 'Mark as Complete'.

#### How to enable Focus Mode in the Learn Page?

Focus Mode will hide the learn page sidebar and header initially. To enable this, go to:

        Masteriyo > Settings > Learn Page > Display

![enable-focus-mode](https://github.com/user-attachments/assets/04dfb033-b38a-4485-8694-50e2982bfecd)

You can choose different options for the Sidebar and the Header. For instance, you can choose show the sidebar and hide the header when the focus mode is enabled.
