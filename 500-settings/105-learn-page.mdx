## Learn Page

### General

![learnpage-general](https://github.com/user-attachments/assets/469a23c7-5f77-446b-b596-5837b814be5c)

- **Logo** - Upload your brand's logo.
- **Auto Load Next Content** - Enable this option to automatically navigate to the next content as we press 'Mark as Complete'.
- **Content Protection [Pro]** -This feature when enabled restricts the ability to right-click, copy text, and open inspect tools on the page to
protect the content.

<Callout>
Please note that, you can easily use the embedded or iframe options in Masteriyo, even with content protection enabled. This means you can allow access to your videos only for specific email addresses, keeping your content private and secure.
</Callout>

- **Lesson Video URL Type** -Select the source type for the lesson video. By default, it uses the Masteriyo Custom Video URL. If you experience video forwarding or skipping issues, consider using the Wordpress Default URL.

### Display

![learnpage-display](https://github.com/user-attachments/assets/0dd7d282-4cae-4a21-8fa4-88222526d1f5)

- **Enable Questions & Answers** - Enable this option to display questions and answers tab in the learn page.
- **Show Sidebar Initially** - When you enable this, the page will navigate to next content as you press 'Mark as Complete'.
- **Enable Lesson Comments** - Enable this option to allow students to comment on lessons.

![Lesson Comments](https://github.com/user-attachments/assets/2fb744f8-6d18-4c3c-8c41-87d163fcef61)

#### How to enable Focus Mode in the Learn Page?

Focus Mode will hide the learn page sidebar and header initially. To enable this, go to:

        Masteriyo > Settings > Learn Page > Display

![learnpage-display-focus-mode](https://github.com/user-attachments/assets/2f4f3605-1332-4492-829b-4b15b4c55b64)

You can choose different options for the Sidebar and the Header. For instance, you can choose show the sidebar and hide the header when the focus mode is enabled.
