### Display

- **Show Search**  
  Toggle to either show or hide search in courses listing page.
- **Courses Per Page**  
  Slide the option to set the total number of courses that you want to show in courses listing page.
- **Courses Per Row**  
  Choose the number of courses that you want to show in a single row.
