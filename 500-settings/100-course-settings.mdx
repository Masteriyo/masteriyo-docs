## Course Page

![course-page-display](https://github.com/user-attachments/assets/bc170f02-258d-4f12-81eb-e7c312215a39)

### Display

- **Courses Per Page** - Set the number of courses to show on a page.
- **Courses Per Row** - Set the number of courses to appear on a row.

### Template

#### Choose Layout

![coursepage-templates](https://github.com/user-attachments/assets/a7d22f6a-99aa-4a59-a898-247b9e252f02)

From this option, you can select different layouts for the Course Archive page. Select a layout from the given option and save. This will be reflected on your Courses page.

#### Enable Custom Template

![coursepage-templates-custom-template](https://github.com/user-attachments/assets/05b9c567-0088-4ee6-b479-646b6a3e7f0d)

You can create an Elementor Template and use it in the course archive page or use the default Masteriyo course archive page template as per your needs. For this, enable the custom template, select the source of the template from 'Template From' and select the template from the next option.

### Styles

![coursepage-styles](https://github.com/user-attachments/assets/a478f1d5-4ead-49bf-bd4d-63413a426bff)

You can customize the various design aspects of the course page from here. You can set button font size, radius, title font size, and course description display side.

The changes you made in the settings are replicated simultaneously on right side where the course page sample is shown.3

### Filtering And Sorting[Pro]

![filtering=sorting](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/50d17bab-2328-4d96-87e6-6d2c7754a7f4)

- **Enable AJAX Behaviour** - This enables behaviour where applying filters will not reload the whole page, instead it will only update the course listing screen.
- **Enable Course Filter** - This allows users to filter courses based on various parameters.
- **Enable Sorting** - This allows users to sort courses in various orders.
