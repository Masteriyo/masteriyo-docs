## Course Page

![course page settings](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/1ab1eec5-f90d-404e-88bb-b79645285c40)

### Display

- **Courses Per Page** - Set the number of courses to show on a page.
- **Courses Per Row** - Set the number of courses to appear on a row.

## Single Course Page

### Display

![Single-Course-Page-Display](https://user-images.githubusercontent.com/75772539/228784702-400353d9-d8fb-438e-8206-af1f2dfd3096.png)

- **Enable Review** - Enable this option to allow the students to review the course

### Related Course

![Single-Course-Page-Related-Course](https://user-images.githubusercontent.com/75772539/228785353-e4e6a536-f76b-47ba-89f9-7f33bd299836.png)

- **Enable** - Option to enable the related course section
- **Related Attribute** - Select either category or instructor as the attribute for the related courses section.
- **Maximum Related Course** - Set the maximum amount of courses to appear on the related courses section.
