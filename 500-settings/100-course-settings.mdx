## Course Page

![course-display](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/ed888e54-9f68-4145-82be-9f40aefe8113)

### Display

- **Courses Per Page** - Set the number of courses to show on a page.
- **Courses Per Row** - Set the number of courses to appear on a row.

### Template

![templates](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/70a881a2-c8ae-4088-9bb3-30b65e8c60d9)

You can create an Elementor Template and use it in the course archive page or use the default Masteriyo course archive page template as per your needs. For this, enable the custom template, select the source of the template from 'Template From' and select the template from the next option.

### Filtering And Sorting

![filtering=sorting](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/50d17bab-2328-4d96-87e6-6d2c7754a7f4)

- **Enable AJAX Behaviour** - This enables behaviour where applying filters will not reload the whole page, instead it will only update the course listing screen.
- **Enable Course Filter** - This allows users to filter courses based on various parameters. 
- **Enable Sorting** - This allows users to sort courses in various orders. 

        Location: Masteriyo > Settings > Single Course Page

## Single Course Page

### Display

![single-course-page-display](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/08d6054f-8d45-4dd9-9025-5d69528a9d60)

- **Enable Review** - Enable this option to allow the students to review the course.
- **Review for Enrolled Users Only** - Enable this to display review tab for enrolled users only.
- **Enable Course Visibilty** - Enable this option to force the users to log in to view paid courses. 

### Related Course

![Single-Course-Page-Related-Course](https://user-images.githubusercontent.com/75772539/228785353-e4e6a536-f76b-47ba-89f9-7f33bd299836.png)

- **Enable** - Option to enable the related course section.
- **Related Attribute** - Select either category or instructor as the attribute for the related courses section.
- **Maximum Related Course** - Set the maximum amount of courses to appear on the related courses section.

### Social Share - **PRO**

![singel-course-page-social-share](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/6061a2ad-b758-4a1c-8eec-e4adf7e9dcbf)

You can choose social sites to allow users to share the course. You can change the given design attirbutes for the social share icons.

### Template

![single-course-templates](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/10e8b234-a788-4287-8dc5-2d252d2a6071)

You can create an Elementor Template and use it in the single course page or use the default Masteriyo course page template as per your needs. For this, enable the custom template, select the source of the template from 'Template From' and select the template from the next option.
