## Display Settings

![account-page-display](https://github.com/user-attachments/assets/814542e4-2924-433d-b603-76524882a144)

- **Enable Order History**: Enable this to display the order history in the Account Page.

![account-page-order-history](https://github.com/user-attachments/assets/4f68fef3-a819-4201-bde0-d14be0610139)

    - **Enable Order Invoice**: Enable this to display the order invoice tab in the Order History tab from where the users can download the Invoice.

    ![account-page-order-history-invoice-button](https://github.com/user-attachments/assets/3ddc10da-9f0b-405d-8b04-baba5431af08)

- **Enable Profile Page**: Enable this to display a profile page tab in the Account page.

![account-page-profile-page](https://github.com/user-attachments/assets/dd9d585d-9daf-45bf-817f-72364f3450a4)

    - **Enable Instructor Apply Button**: Enable this option to display a 'Apply for Instructor' button on the Profile Page tab.
    - **Enable Edit Profile Button**: Enable this option to display a Edit Profile button on the Profile Page tab in the Account Page.

    ![account-page-profile-page-buttons](https://github.com/user-attachments/assets/763b652d-7f6f-4e6e-8a0d-ea5238c81b25)
