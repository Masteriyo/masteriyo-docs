Once you install and activate the Masteriyo LMS plugin, you will be able to access the Masteriyo Dashboard. Masteriyo Dashboard is a place from where you can get access to various parts of the plugin like add-ons, settings, documentation, and support. Let us check what does this section really consists of.

![masteriyo-dashboard](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/cd000f59-5285-4d8e-906c-974927b3396f)

As you can see, there are various tabs on the dashboard page. These tabs are described as below:

### Dashboard

This is the tab where you land at first. From here, you can access Masteriyo video tutorials, community, documentation, etc. Also, if you have any trouble with the plugin, you can directly create a support ticket from here. Moreover, if you see any features missing, you can request for the feature as well.
Don't forget to leave a review for the plugin from the **Submit a Review** option.

### Addons

In this section, you will find all the free and premium add-ons of the Masteriyo plugin.

![dashboard-addons](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/4746a9d6-98fb-43a1-99b3-b1bfea4063ac)

If you haven't subscribed to a premium plan, you can make a purchase via this page as well. The list comes with advance sorting and searching options to make your experience better and faster. Also, you can check the active and inactive add-ons from this same spot.

### settings

The next tab is \*_Settings_. Click on this tab and you will be redirected to the Masteriyo settings page from where you can set up the plugin as per you requirement.

        [Wanna learn more about Settings](https://docs.masteriyo.com/settings)

### Help

![dashboard-help](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/7138858a-3e50-4bf3-8f21-f8a7a28605b9)

This tab consists of information required to help the users adapt to the Masteriyo plugins. You can check all the available shortcodes and smart tags from here as well as create a support ticket and view documentation.

### Free Vs Pro

![dashboard-free-vs-pro](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/fbe94124-247b-4f2b-a56d-ba189314e79b)

In this section, we have compared the free and pro version of the plugin based on the features they have. This way it will be easier for our users to decide whether they need the upgrade or not.

Also, scroll to the bottom of the page and click on the **Get Masteriyo Pro Now** button for upgrading to the premium version of the plugin.

### Other Products

![dashboard-other-products](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/2ebb3d51-ab28-4a3e-ba9c-69714eb4a40f)

Here, you can see other products from our company. We have various other plugins and themes as well which you can check.

### Extra options in the Navigation Bar

![dashboard-nav-bar](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/f71e2694-6f33-4fa7-91e2-77cf5587b543)

On the right-hand side of the navigation bar, you will see the version number, option to upgrade to pro, and latest updates from our side.
