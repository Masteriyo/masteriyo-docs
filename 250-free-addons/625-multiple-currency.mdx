This add-ons allows you to sell the same product in different currencies for different countries.

## Installation

- Activate the [Multiple Currency add-on](https://docs.masteriyo.com/getting-started/activate-masteriyo-addons)

## Configuration

### Create Pricing Zone/Zones

Once you have activated the Multiple Currency add-on, go to **Masteriyo > Multiple Currency**.

![multiple-currency-all-currency](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/afb27f11-2371-4441-834a-f4b50249e2ea)

- Click on the **Add New Pricing Zone** button.

![add-mew-pricing-zone](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/91da528f-d878-40d4-8e37-4354f9085829)

You will then see the following screen.

![new-pricing-zone-details](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/f3c01de6-8e9c-46e6-bc8d-38b256aa8c0a)

1. **Zone Name**: Provide a relevant name for the zone you are adding.
2. **Exchange Rate**: You will need to enter the exchange rate manually.
3. **Countries**: Select the countries that are withing the zone you are creating.
4. **Currency**: Choose a currency for the zone. The customers from these countries will then be able to see the price and pay it with the selected currency.

![pricing-zone-created](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/4611ecda-b0c4-429b-b446-65859048b7fd)

- Click on the **Create** button.

The pricing zone will be added. You can update it anytime you want.

### Pricing Zone Global Settings

        Location: Masteriyo > Multiple Currency > Settings

![global-settings-for-pricing-zones](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/bfc230e1-eae1-477c-9605-61d30aa564cb)

These settings will be applied for all the pricing zones you create. There are two settings for the pricing zones.

1. **Enable Geolocation Integration**: The integration with MaxMind Geolocation helps you to enable accurate country detection for providing localizes currency support to the customers. You will need the MaxMind License Key for this settings.
2. \*_Enable Test Mode_: Select a country, within the Pricing Zones you have set, for the test mode.

### Enable Multiple Currency in Courses

- Go to **Courses - Edit a Course**.
- Go to **Settings > Pricing**. From here, you can enable Multiple Currency.

        Note: If you are using Pro version, this option is available only for One Time Fee

![enable-multiple-currency-in-course](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/4a435dd0-a114-43cf-a6ec-15b58e8a67b3)

The pricing method can be set within the Multiple Currency settings.

![price-method](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/a53f5b84-374b-4689-add4-91e9f84e11bc)

1. **Calculate prices by the exchange rate**: This will calculate the price as per the exchange rate you have set in the Pricing Zone settings.
2. **Set price manually**: From here, you can discard the exchange rate option and set manual prices.
