The Oxygen Builder plugin is a powerful and versatile tool that has revolutionized the way websites are designed and built on the WordPress platform. With its user-friendly interface and extensive customization options, Oxygen Builder empowers web developers and designers to create stunning and responsive websites with ease When integrated into Masteriyo, it can provide you with the tools to create engaging and interactive e-learning content for your students.

**Requires**

- Oxygen plugin to be installed and activated.
- [Oxygen Integration add-on activated.](https://docs.masteriyo.com/getting-started/activate-masteriyo-addons)

# Setup and Configuration

Once you have installed the Oxygen plugin and activated the Oxygen Integration add-on from Masteriyo, follow the steps below.

- Go to **Dashboard > Pages > Add New**.
- Give the page's title and click on **Publish**. 

Now, you can see the **Edit with Oxygen** button. Click on this button to get started with the Oxygen Builder. 

![edit-with-oxygen](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/7854ff64-d3bb-4fd9-a953-7c2b0e7725fd)

You will now be able add elements on your page with the Oxygen Builder.

![add-elements](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/84e505a5-c20f-4046-b8d9-a9e601dcdf4c)

Click on the '+' button on the top-left, search for the course elements and add the desired element. There are two options available. 

![add-elements](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/84e505a5-c20f-4046-b8d9-a9e601dcdf4c)

1. **Course List**
2. **Course Categories**

Let us add the Course List element for sample. 

![course-list-element](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/4af4c3f4-35ad-4921-b1ad-8486950aa984)

After you add this element, you can see the lists of all the courses on your site. There are numerous option on the left hand side provided by the Oxygen builder and Masteriyo for you to customize the appearance. 

### Filtering the Courses on the list

![filter-category](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/ed7dcc16-6791-42be-9ff4-63df5166d7ad)

You can also filter the courses to appear on the list from the Filter tab. You will need the categories/instructors IDs to filter. You can get the category ID from the following location.

        Masteriyo > Categories > Click on the desired category

Now, on the URL box, you will find the category on the last part of the URL.

![categories-id](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/d00f2b58-c726-4642-9028-ef15b9a4d97d)

You can write this ID on the available filter options. Multiple IDs can be written separated by IDs. 

In this same way, you can set up the Course Category element as well.
