Equip your Bricks editor with Masteriyo elements. Add components like course lists and categories to any page/post.

**Requires**

1. [Masteriyo LMS](https://masteriyo.com/pricing/)
2. [Bricks Theme](https://bricksbuilder.io/pricing/)
3. [Bricks Integration Add-on](https://docs.masteriyo.com/getting-started/activate-masteriyo-addons)

## Setup and Configuration

After the activation, you can create a page with Bricks builder from Pages > Add New Page > Edit with Bricks.

![edit-with-bricks](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/bad62ddb-5917-4f2b-9745-46f25746dc52)

Now, add a container of your need. On the left, you will find all the Bricks blocks listed. Search for the following.

![courses-blocks](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/7d79cd13-ef4f-4a10-bea2-70aaf2804b7a)

1. Courses: All the courses you have created with Masteriyo will be available.
2. Course Categories: List of the course categories created with Masteriyo.

### Course Categories

Drag the Course Categories blocks into the container.

![course-categories-dragged](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/e16435d3-bc43-43ba-b2b4-e3499970594e)

Now, you can configure content and styling by clicking on the categories block(on the page editor), as per your requirements.

### Courses

Similarly, drag the courses blocks to show all the courses you have created with the Masteriyo plugin.

![courses-blocks](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/3714e947-c615-4354-845b-58b2ae29e19a)

Click on the courses block on the page editor to find the options that can be configured. It includes various options like how many courses to display per page, layout, show/hide difficulty badge, show/hide categories/authors, pagination and many more.

### Bricks temmplate for Course Page and Single Course Page

- Go to **Bricks > Templates**.

Here, you will be able to see two templates from Masteriyo in default. You can edit these templates or create your own custom template.

To create a custom template, follow the steps below.

- Click on **Add New**.

![add-new-template](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/06c2688f-b534-429c-a6f0-b66c181d6d6d)

On the **Select template type** you need to select either Masteriyo Single Course or Masteriyo Course Archive as per your requirements.

Now, click on **Edit with Bricks**. From the editor, you can edit the parts of the Masteriyo Course or Course Archive to suit your needs.

Once done, go to the following location:

        Masteriyo > Settings > Course Page or Single Course Page > Templates

- Enable custom template.
- Choose **Bricks** on the **Template from** option.

![choose-bricks-template](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/08d912e5-ad71-41dc-bdb0-46665d79f9e6)

- From the dropdown of the Template option, you can choose the template you have created.
