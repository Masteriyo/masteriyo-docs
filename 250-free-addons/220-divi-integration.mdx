**Requires**

- Divi theme to be installed and activated.
- [Divi Integration add-on activated.](https://docs.masteriyo.com/getting-started/activate-masteriyo-addons)

# Setup and Configuration

Once you have installed the Divi Theme and activated the Divi Integration add-on from Masteriyo, follow the steps below.

- Go to Pages > Add New 

![create-new-course-page-with-divi](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/a20004e8-4b96-42da-8779-af80103a6639)

- Provide the name for the page and click on **Use Divi Builder**.

Now, you will see the Divi Builder screen from where you will be building the course/courses page. You can choose to build the page from scratch, choose a template or import an existing page. Then, select the number of rows you want on your page. After this, you will have to choose the Masteriyo Modules from the available Divi Modules.

![masteriyo-divi-element](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/6b6205d9-2619-40cf-ac5d-fba0d3a8d714)

There are two Modules you can choose from. They are:
1. Masteriyo Course Categories
2. Masteriyo Course List

Select the one you need and you will see a popup to design the selected segment. 

![design-masteriyo-divi-modules](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/3da90290-0b82-431e-b0f9-8e417795ca3f)

On the Content tab, you can set number of categories and columns. Similarly, there are toggle buttons from which you can enable/disable options like include sub-categories, show/hide thumbnail, details, title, course count, etc.
On the Design tab, you will find attributes to desgin the Modules as per your need. 

        If the popup is closed, you can double click on the row to bring the popup back.  

These options will apply for both of the Modules.
