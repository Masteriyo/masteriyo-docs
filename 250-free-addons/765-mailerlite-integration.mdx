Now you can integrate MailerLite with Masteriyo LMS to streamline your email Marketing. 

## Installation

- Activate the [MailerLite add-on] (https://docs.masteriyo.com/getting-started/activate-masteriyo-addons)

## Getting MailerLite API key

- [Log in to your MailerLite account](https://accounts.mailerlite.com/). If you donâ€™t have an account, [sign up here](https://www.mailerlite.com/signup)
- After logging in, navigate to Integrations to generate your API

![mailerlite-generate-token](https://github.com/user-attachments/assets/895ec29e-c3c5-496f-975c-df42e681fce1)

- Copy the generated API key to a clipboard.

![mailerlite-api-key](https://github.com/user-attachments/assets/1a174591-b652-4378-9ea2-54ab135b3fdf)

### Adding API key to Masteriyo to connect MailerLite

- Go to **Masteriyo > Settings > Integration > MailerLite Integration**.

![mailerlite-integration-settings](https://github.com/user-attachments/assets/add57eef-dcde-4eb1-9d62-f7b2dc969fc9)

- Add the generated key and click on the connect button.

You will then get the account settings for the connected Brevo account, as shown below.

![mailerlite-account-settings](https://github.com/user-attachments/assets/9da61ce3-40a1-4a34-94e4-042b8215facd)

- **Forced Subscription**: Enable this option to automatically subscribe the users to your email list.
- **Consent Message**: This message will be displayed to the users in the registration form when asking for their mail subscription consent. 
- **Groups**: Select a group from the dropdown field to subscribe the users to.

Once done, click on the **Save Settings** button.

Now, you can see the checkbox on the Masteriyo registration form on your site. 

![registration-form-masteriyo-mailerlite](https://github.com/user-attachments/assets/ab6bea54-a6c7-4adc-bad9-55d5d7fe4781)
