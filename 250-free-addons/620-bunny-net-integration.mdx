## Installation

- Install [Masteriyo Pro](https://docs.masteriyo.com/getting-started/installation#installing-masteriyo-lms-pro-version)
- Activate the [Bunny Net Integration add-on](https://docs.masteriyo.com/premium-addons/activate-masteriyo-premium-addons)

## Configuration

### Creating a Video Library in Bunny Net

- Log in or sign up to [Bunny Net CDN](https://dash.bunny.net/auth/login?pk_buttonlocation=menu).
- Go to Stream

![bunny-net-stream](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/e848c892-fc3b-4c14-bfd4-617c43445050)

- Click on **Add Video Library**.
- Enter the necessary details and create a Video library.
- Upload Videos to the library.

![upload-video](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/09e69796-9b40-4cd5-adcb-87cf3dd0864c)

- Click on the uploaded video to get the video information.
- From the given information, we will need the **Embed** HTML code.

![embed-code](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/910f404d-cab2-42a8-9554-8e750e10b88e)

- Copy the code into a clipboard.

### Adding Bunny Net Video into Masteriyo courses

- Go to **Masteriyo > Courses > Edit a course**.
- Go to **Builder > Add New Section**.
- In the added section, you can add a new Lesson.
- Given the Lesson Name, and on the **Video Source** option, choose **Bunny Net**.

![video-source-bunny-net](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/0ba2c9b4-274b-43ca-b0af-9fd106b8b84a)

- This will let you enter the above copied HTML code from Bunny Net. You will then be able to see the video on the Course Editor as shown on the image.
