User Registration Integration addon integrates with the User Registration Plugin to maximize the registration process.

**Requires**

1. [Masteriyo LMS](https://masteriyo.com/pricing/)
2. [User Registration Plugin](https://wordpress.org/plugins/user-registration/)
3. [User Registration Add-on](https://docs.masteriyo.com/getting-started/activate-masteriyo-addons)

## Setup and Configuration

Once you have installed and activated the above plugins and add-ons, you can configure this integration by following the steps below.

## Create Registration form with User Registration Plugin

First of all, you will have to create a registration form with User Registration plugin. To do this, follow the steps below.

- Go to **User Registration > Add New**
- Add required form fields.

![user-registration-add-new](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/e436fc14-cd4e-441e-9da0-f478958e5e4c)

- Go to **Forms Settings > General**. Here, you need to set the User Role either to be Masteriyo Student or Masteriyo Instructor as per your need. 

![set-user-role](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/88a40e6f-2631-49ad-a58f-5308836f3e02)

- You can also set the User Approval option like admin approval, email confirmation, auto-approval, and auto/manual login. 
- Click 'Update form'.

## Set registration form for Students and Instructor in Masteriyo Settings

Now, go to **Masteriyo > Settings > Integrations > User Registration Integration**

![user-registration-settings-in-masteriyo](https://github.com/user-attachments/assets/e8dd89a1-f510-4ed9-bcbb-d6aa6467f34d)

Here, you can enable Custom Student/Instructor Registration forms created in the User Registration plugin. 

![select-form](https://github.com/user-attachments/assets/4464da26-ee03-4ba4-abc2-39e18eebfaef)

Now, the User Registration form you selected will be used for registering students or instructors on your site.
