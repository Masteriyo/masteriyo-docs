User Registration Integration addon integrates with User Registration Plugin to maximize registration process.

**Requires**

1. [Masteriyo LMS](https://masteriyo.com/pricing/)
2. [User Registration Plugin](https://wordpress.org/plugins/user-registration/)
3. [User Registration Add-on](https://docs.masteriyo.com/getting-started/activate-masteriyo-addons)

## Setup and Configuration

Once you have installed and activated the above plugins and add-ons, you can configure this integration by following the steps below.

## Create Registration form with User Registration Plugin

First of all, you will have to create registration form with User Registration plugin. To do this, follow the steps below.

- Go to **User Registration > Add New**
- Add required form fields.

![user-registration-add-new](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/e436fc14-cd4e-441e-9da0-f478958e5e4c)

- Go to **Forms Settings > General**. Here, you need to set User Role either to be Masteriyo Student or Masteriyo Instructor as per your need. 

![set-user-role](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/88a40e6f-2631-49ad-a58f-5308836f3e02)

- You can also set the User Approval option like admin approval, email confirmation, auto approval, auto/manual login. 
- Click 'Update form'.

## Set registration form for Student and Instructot in Masteriyo Settings

Now, go to **Masteriyo > Settings > Advanced > User Registration Integration**

![user-registration-settings-in-masteriyo](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/00be8afd-d42e-4a89-b554-f5c166ffd31c)

Here, you can enable Custom Student/Instructor Registration forms created in User Registration plugin. 

![select-form](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/fa80cd18-0bf4-49e9-bb51-76d78e93f924)

Now, the User Registration form you selected will be used for registrating student or instructors on your site.
