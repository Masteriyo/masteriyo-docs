# General Settings

### Styling
- **Primary Color**  
Set a Color to match your brand or site. This color reflects on buttons, links and few other elements.  
**Note**: This color does not affect the colors in the course learning page.

### Courses Page
- **Show Search**  
Toggle to either show or hide search in courses listing page.
- **Courses Per Page**  
Slide the option to set the total number of courses that you want to show in courses listing page.
- **Courses Per Row**  
Choose the number of courses that you want to show in a single row.
