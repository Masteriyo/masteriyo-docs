# Technical Requirements

### General Configuration Recommendations
- **WP version**: 5.0 or higher
- **PHP version**: 7.0 or higher
- **MySQL version**: 5.6 or higher

### PHP Configuration Requirements
- **Post Max Size**: 64 MB
- **Memory Limit**: 256 MB
- **Time Limit**: 300
- **Max Upload Size**: 64 MB
- **WP Remote Get**: enabled

### Hosting Requirements
Make sure your hosting supports WordPress minimum requirements: [https://wordpress.org/about/requirements/](https://wordpress.org/about/requirements/)
