All of the code snippets for modifying existing features can be found below. Check out [this link](https://docs.masteriyo.com/faqs/custom-code-snippet) if you do not know where to add custom code snipeets to your site.

### Change Default Order Status For Ofline Payment

```php
add_filter( 'masteriyo_offline_process_payment_order_status', function( $status ) { 
    return 'completed'; 
});
```

### Change Courses Featured label [PRO]

```php
add_filter( 'masteriyo_pro_course_featured_text', function( $text ) {
    return 'New Featured';
});
```
