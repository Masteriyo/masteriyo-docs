The Event Calendar addon by Masteriyo shows the monthly schedule of assignments, quizzes, lessons, and Zoom meetings.

## Setup

1. Install [Masteriyo LMS Pro plan or higher](https://docs.masteriyo.com/getting-started/installation#installing-masteriyo-lms-pro-version).
2. Activate the [Event Calendar add-on](https://docs.masteriyo.com/getting-started/activate-masteriyo-addons).

## Configuration

Once you install and activate the Event Calendar add-on, a new tab will be added to the students' account page. Go to the **Calendar** tab on the Account page to view all the upcoming events.

![calendar-tab](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/587f4911-0817-44dd-a290-adf1efddb56b)

Click on each date to see the events for that day. You can also select a month and the year as you need them. Events include assignment dates, Zoom meetings(if integrated), etc.
