Event Calendar addon by Masteriyo shows monthly schedule of assignments, quizzes, lessons, and Zoom meetings.

## Setup

1. Install [Masteriyo Pro](https://docs.masteriyo.com/getting-started/installation#installing-masteriyo-lms-pro-version)
2. Activate the [Event Calendar add-on](https://docs.masteriyo.com/premium-addons/activate-masteriyo-premium-addons)

## Configuration

Once you install and activate the Event Calendar add-on, a new tab will be added in the Account page of the students.
