The Paid Membership Pro Integration add-on lets you integrate the Paid Membership Pro plugin with the Masteriyo to enhance your ability to monetize and manage course access within the Masteriyo learning environment.

## Prerequisites:

1. To use this add-on, install and activate the [Paid Membership Pro](https://wordpress.org/plugins/paid-memberships-pro/) plugin. 
2. Install [Masteriyo LMS Pro plan or higher](https://docs.masteriyo.com/getting-started/installation#installing-masteriyo-lms-pro-version)
3. Activate the [Paid Membership Pro Integration Add-on](https://docs.masteriyo.com/getting-started/activate-masteriyo-addons)

## Configuration

Once you have installed and activated this add-on, go to the following location.

        **Masteriyo > Courses > Edit a Course > Settings > Paid Membership Pro**

![paid-memebership-pro-settings](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/d0f5c1a2-80b6-4714-bb74-e158222380e8)

There are a few options to set here. They are as follows:

1. **Required Membership Levels** - You can select a Membership level that you have created in the Paid Membership Pro plugin.
2. **Enrollment Button URL** - Add the URL of the page that you have created to enroll the users to the plan created via the Paid Membership Pro plugin. 

Apart from the above options, you can also assign the Masteriyo courses to each of the Membership levels. For this, go to the following location.

       **Memberships > Settings > Select a Level > Other Settings**

![membership-level-other-settings](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/22e07c01-0bc2-4b13-bef3-73362c63ec58)

You can choose the required courses for the level from the available dropdown. You can select multiple courses.
