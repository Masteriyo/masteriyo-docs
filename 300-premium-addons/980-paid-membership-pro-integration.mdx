The Paid Membership Pro Integration add-on lets you integration the Paid Membership Pro plugin with the Masteriyo to enhance your ability to monetize and manage course access within the Masteriyo learning environment.

## Setup

You will need to install and activate the [Paid Membership Pro](https://wordpress.org/plugins/paid-memberships-pro/) plugin to use this add-on. 

![activate-paid-membership-pro-addon](https://user-images.githubusercontent.com/75772539/235430273-1ffad56a-c3ba-4fab-a016-28265e2c05f6.png)

1. Install [Masteriyo Pro](https://docs.masteriyo.com/getting-started/installation#installing-masteriyo-lms-pro-version)
2. Activate the [Paid Membership Pro Integration Add-on](https://docs.masteriyo.com/getting-started/activate-masteriyo-addons)

## Configuration

Once you have installed and activated this add-on, go to the following location.

        Masteriyo > Courses > Edit a Course > Settings > Paid Membership Pro

![paid-memebership-pro-settings](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/d0f5c1a2-80b6-4714-bb74-e158222380e8)

There are few options to set here. They are as below:

1. **Required Membership Levels** - You can select a Membership level that you have created in the Paid Membership Pro plugin.
2. **Enrollment Button URL** - Add the URL of the page that you have created to enroll the users to the plan created via the Paid Memebership Pro plugin. 

Apart from the above options, you can also assign the Masteriyo courses to each of the Membership levels. For this, go to the following location.

        Memberships > Settings > Select a Level > Other Settings

![membership-level-other-settings](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/22e07c01-0bc2-4b13-bef3-73362c63ec58)

You can choose the required courses for the level from the available dropdown. You can select multiple courses.
