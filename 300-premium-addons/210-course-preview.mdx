This feature allows students to have a sneak peek at lessons prior to enrolling in a course.

## Prerequisite:

1. Active installation of [Masteriyo Basic plan or higher](https://docs.masteriyo.com/getting-started/installation#installing-masteriyo-lms-pro-version).
2. Activate installation of [Course Preview Add-on](https://docs.masteriyo.com/getting-started/activate-masteriyo-addons)

## Configuration

Once you have enabled the Course Preview add-on, go to **Masteriyo > Courses > Edit a Course**. Now, edit a lesson for which you want to enable the preview.

![enable-preview-button](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/a6cbdd49-2530-4f2f-bfef-d9cc83c17498)

In the lesson editor, you will see the button to enable the preview. Enable the preview from this button.

Once you have enabled this, you can see a button on the Course Page > Curriculum > Lessons.

![preview-button](https://github.com/Masteriyo/masteriyo-docs/assets/75772539/c9d0e17c-86d6-408d-afae-f816d3e537f3)

The students can preview the lesson via this button.
