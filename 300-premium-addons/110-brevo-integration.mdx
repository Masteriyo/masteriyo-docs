Seamlessly integrate Brevo with Masteriyo LMS for efficient and automated email marketing within your learning management system.

## Setup and Configuration

**Note: This add-on is available from the GROWTH plan only.**

- Install [Masteriyo Pro](https://docs.masteriyo.com/getting-started/installation#installing-masteriyo-lms-pro-version)
- Activate the [Brevo Integration add-on](https://docs.masteriyo.com/premium-addons/activate-masteriyo-premium-addons)

### Getting Brevo API key

- [Log in to your Brevo account](https://login.brevo.com/). If you donâ€™t have an account, [sign up here](https://onboarding.brevo.com/account/register).
- After logging in, navigate to your API keys section, from your account icon on the top-right, to generate a new API key. [Go to API Keys]( After logging in, navigate to your API keys section to generate a new API key. Go to API Keys)

![brevo=smtp-and-api](https://github.com/user-attachments/assets/a4f25ea3-c816-40c6-b0f4-b908bf68df8b)

- Copy the generated API key to a clipboard.

![brevp-generated-api-key](https://github.com/user-attachments/assets/e3079853-3d5c-44b6-896a-ee7b763f747b)

### Adding API key to Masteriyo for connecting Brevo

- Go to **Masteriyo > Settings > Integration > Brevo**.

![brevo-integration-settings](https://github.com/user-attachments/assets/2cbc476c-a163-4c1a-8048-fdbab01d0806)

- Add the generated key and click on the Connect button.

You will then get the account settings for the connected Brevo account as shown below.

![brevo-account-settings-in-masteriyo](https://github.com/user-attachments/assets/95282d9e-96e0-4508-b451-38bb549f2cd0)

- **Forced Subsciption**: Enable this option to automatically subscribe the users to your email list.
- **Consent Message**: This message will be displayed to the users in the registration form when asking for their mail subscription consent.
- **Lists**: Select a list from the dropdown field to subscribe the users to.

Once done, click on the **Save Settings** button.
