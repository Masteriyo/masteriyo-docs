This feature allows you to add materials for your courses that visitors need to download.

### How to Use?

![activating-course-attachment-in-masteriyo](https://user-images.githubusercontent.com/24367265/195078870-70b64d1b-8172-4ff7-90d5-b7c4a6d3bd0d.png)

1. Install [Masteriyo Pro](https://docs.masteriyo.com/getting-started/installation#installing-masteriyo-lms-pro-version)
2. Activate the [Course Attachments Add-on](https://docs.masteriyo.com/getting-started/activate-masteriyo-addons)
3. Go to your **Dashboard**, navigate **Masteriyo**->**Courses**
4. **Select** and **Edit** the course where you want to add an attachment
5. Click on **Attachments** from the right side and upload the documents
6. Click the **Update**

![displaying-course-attachment-with-masteriyo-plugin](https://user-images.githubusercontent.com/24367265/195078918-1497a23f-66b6-4eba-83a0-ac26994095dd.png)

<Callout>
  {" "} 
**Note:** Only users enrolled in the course can access the attachments.
</Callout>

### Supported File Types

Masteriyo LMS allows you to upload course materials from your media library or directly from your device. In addition to WordPress-supported file types, the following additional formats are supported:

#### Application Files

| Format               | MIME Type                                                                                     | Extensions   |
|----------------------|-----------------------------------------------------------------------------------------------|--------------|
| PDF                  | `application/pdf`                                                                             | .pdf         |
| ZIP Archive          | `application/zip`                                                                             | .zip         |
| Microsoft Word       | `application/msword`<br/>`application/vnd.openxmlformats-officedocument.wordprocessingml.document` | .doc<br/>.docx |
| Microsoft PowerPoint | `application/vnd.ms-powerpoint`<br/>`application/vnd.openxmlformats-officedocument.presentationml.presentation` | .ppt<br/>.pptx |
| Microsoft Excel      | `application/vnd.ms-excel`<br/>`application/vnd.openxmlformats-officedocument.spreadsheetml.sheet`     | .xls<br/>.xlsx |

## Image Files

| Format | MIME Type     | Extensions  |
|--------|---------------|-------------|
| JPEG   | `image/jpeg`  | .jpg, .jpeg |
| PNG    | `image/png`   | .png        |
| GIF    | `image/gif`   | .gif        |


## Video Files

| WMV | `video/x-ms-wmv` | .wmv |
